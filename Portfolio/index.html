Color Game

Juego web simple e intuitivo en el que el usuario ve un código RGB (por ejemplo, rgb(255, 99, 71)) y debe adivinar a cuál de los cuadrados de color corresponde. Al hacer clic en una opción:

  • Si acierta, todos los cuadros cambian al color correcto y aparece un mensaje de felicitación.
  • Si se equivoca, el cuadro desaparece y puede seguir intentando.
La interfaz es limpia, colorida y responsive, pensada para brindar una experiencia lúdica y visualmente atractiva. Ideal para quienes quieran poner a prueba su percepción del color de forma rápida y entretenida.

To Do List

Aplicación web sencilla y funcional para organizar tareas diarias. El usuario puede:

  • Agregar tareas mediante un campo de texto
  • Marcar como completadas haciendo clic sobre ellas (cambia el estilo para indicar que está hecha).
  • Eliminar tareas individualmente con un botón de borrar.
El diseño es minimalista y responsive, pensado para que cualquier persona pueda anotar y gestionar sus pendientes de forma rápida y clara, desde cualquier dispositivo.

EventFlow

Aplicación web para gestionar eventos personales de forma simple y ordenada. El usuario puede:

  • Agregar eventos con título, fecha y hora.
  • Visualizar los eventos en una lista clara y cronológica.
  • Eliminar eventos fácilmente cuando ya no son necesarios.
Cuenta con un diseño limpio y amigable, pensado para facilitar la planificación diaria. Ideal para quienes buscan una herramienta rápida y accesible para organizar sus actividades.
